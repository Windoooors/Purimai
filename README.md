# Purimai

#### Introduction
A maimai simulator dedicated to recreating the old time glory of maimai FiNALE, supporting 2simai.
Currently this project is still working in progress.

#### Usage
1. Launch the game.
2. Put 2simai chart folders into /sdcard/Android/data/com.Setchin.Purimai/files/Charts on Android or Purimai/Charts on iOS.
3. Launch the game again and enjoy.
